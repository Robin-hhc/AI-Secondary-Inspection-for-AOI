# 工业AI质检系统用户手册

## 1. 系统概述

工业AI质检系统是基于主动学习和异常检测的车灯制造产线AOI辅助系统。系统通过PatchCore算法进行异常检测,利用主动学习策略筛选不确定样本供人工标注,持续优化模型性能。

### 1.1 核心功能

- **异常检测**: 基于PatchCore的无缺陷样本异常检测
- **主动学习**: 自动筛选不确定样本供人工标注
- **模型更新**: 利用标注数据持续优化模型
- **多产品支持**: 支持多产品型号动态切换
- **Web标注终端**: 提供友好的标注界面

### 1.2 系统架构

```
AOI设备 → 图像采集 → 推理引擎 → 主动学习 → 标注终端 → 模型更新
```

## 2. 快速开始

### 2.1 系统要求

- 操作系统: Ubuntu 20.04/22.04 或 Windows 10+
- Python: 3.8+
- Docker: 20.10+
- GPU: NVIDIA GPU (可选,推荐)

### 2.2 安装步骤

#### 方式一: Docker部署 (推荐)

```bash
# 1. 克隆项目
git clone <repository_url>
cd aoi_quality_inspection

# 2. 启动系统
bash scripts/start.sh

# 3. 访问系统
# Web终端: http://localhost:5000
```

#### 方式二: 手动安装

```bash
# 1. 安装Python依赖
pip install -r requirements.txt

# 2. 初始化数据库
python scripts/init_db.py

# 3. 启动Web服务
python web_terminal/backend/app.py
```

## 3. 功能使用

### 3.1 登录系统

1. 打开浏览器访问 `http://localhost:5000`
2. 输入用户名和密码
3. 点击"登录"按钮

### 3.2 标注操作

#### 3.2.1 查看待标注样本

1. 登录后进入"标注"页面
2. 系统自动显示待标注样本列表
3. 每个样本显示:
   - 图像预览
   - AI异常分数
   - AI判定结果
   - 采集时间

#### 3.2.2 执行标注

1. 选择待标注样本
2. 查看图像和AI判定结果
3. 选择标注结果:
   - **正常**: 样本为合格品
   - **缺陷**: 样本为不良品
4. (可选) 选择缺陷类型
5. (可选) 添加备注
6. 点击"提交"按钮

#### 3.2.3 快捷键

- `N`: 标记为正常
- `D`: 标记为缺陷
- `Enter`: 提交标注
- `→`: 下一个样本
- `←`: 上一个样本

### 3.3 统计查看

#### 3.3.1 统计概览

进入"统计"页面查看:
- 总样本数
- 已标注数
- 正常/缺陷数量
- 不确定样本数

#### 3.3.2 性能趋势

查看性能趋势图表:
- 准确率趋势
- 标注进度
- 模型性能

### 3.4 模型管理

#### 3.4.1 查看产品型号

进入"模型管理"页面查看所有产品型号

#### 3.4.2 切换产品型号

1. 选择目标产品型号
2. 点击"切换"按钮
3. 系统自动加载对应特征库

#### 3.4.3 触发模型更新

1. 进入"模型管理"页面
2. 点击"触发更新"按钮
3. 系统利用新标注数据更新模型

## 4. 系统配置

### 4.1 配置文件

配置文件位于 `config/config.yaml`

### 4.2 主要配置项

```yaml
# 推理引擎配置
inference:
  threshold:
    anomaly_threshold: 0.5  # 异常判定阈值
    uncertainty_margin: 0.1  # 不确定性边界

# 主动学习配置
active_learning:
  sampling:
    max_samples: 100  # 最大采样数量

# 模型更新配置
model_updater:
  trigger:
    min_samples: 50  # 最小更新样本数
```

### 4.3 修改配置

1. 编辑 `config/config.yaml`
2. 修改配置项
3. 重启系统

## 5. 常见问题

### Q1: 系统启动失败?

**A**: 检查以下项:
- Docker是否正常运行
- 端口5000是否被占用
- 数据目录权限是否正确

### Q2: 推理延迟过高?

**A**: 尝试以下优化:
- 启用GPU加速
- 减小批量大小
- 使用TensorRT优化模型

### Q3: 标注界面无法显示图像?

**A**: 检查:
- 图像路径是否正确
- 图像文件是否存在
- 浏览器控制台错误信息

### Q4: 模型更新失败?

**A**: 检查:
- 是否有足够的标注数据
- 特征库路径是否正确
- 磁盘空间是否充足
